# 💻 Hacker Terminal Simulation

A fun JavaScript project that simulates a **hacker terminal**. The program displays fake hacking messages one by one with random delays while animating blinking dots (`.`) to create a realistic terminal effect.

> ⚠️ **Disclaimer:** This project is created **for educational and entertainment purposes only**. It does **not** perform any real hacking or access any user data.

---

## ✨ Features

- 🎲 Random delay between messages for a realistic effect
- ⏳ Animated blinking dots (`.` → `..` → `...`)
- 🔄 Uses **Promises** and **Async/Await**
- 📜 Dynamically creates and appends messages to the webpage
- 🎯 Demonstrates asynchronous JavaScript concepts in a fun way

## 🚀 How It Works

1. The program starts the hacking simulation.
2. Each message appears after a **random delay (1–7 seconds)**.
3. While waiting, the latest message displays animated dots to simulate processing.
4. The process continues until all messages have been displayed.
5. The animation stops after the final message.
